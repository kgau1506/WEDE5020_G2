## Project Overview

Skin Syntax is a skincare website designed to provide users with information about skincare products, skincare routines, and skin type guidance. The website aims to promote healthy skin practices while offering specialized skincare solutions for individuals with different skin concerns, including vitiligo and albinism.

## Website Purpose

The purpose of Skin Syntax is to:

* Educate users about different skin types.
* Provide skincare product recommendations.
* Promote healthy skincare routines.
* Offer skincare solutions for diverse skin needs.
* Allow users to submit enquiries and contact the business.
